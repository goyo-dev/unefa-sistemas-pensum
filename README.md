# 🎓 Pénsum Ingeniería de Sistemas - UNEFA

¡Bienvenido al sistema de gestión académica diseñado para los estudiantes de Ingeniería de Sistemas de la UNEFA!

Esta herramienta no es solo una visualización del pensum; es un **centro de mando personal** para planificar tu ruta académica con precisión matemática.

## ✨ Características Principales

- **Árbol de Prelaciones Interactivo:** Visualiza la ruta crítica de tu carrera. Identifica al instante qué materias están bloqueadas y cuáles son las llaves maestras (`🔥`) para desbloquear los siguientes semestres.
- **Calculadora de Índice Ponderado:** Olvida los cálculos manuales. Registra tus notas y la aplicación calculará en tiempo real tu índice ponderado basándose en el peso (UC) de cada asignatura.
- **Trazado de Ruta (Pathfinding):** ¿Quieres saber qué necesitas cursar para ver una materia específica en el futuro? Haz clic sobre cualquier asignatura bloqueada y la app resaltará el camino exacto de requisitos previos.
- **Progreso Visual:** Barra de progreso automático que te indica cuántas Unidades de Crédito (UC) llevas acumuladas respecto a la meta de 214 UC para pasantías.
- **Modo Oscuro (Dark Mode):** Diseño optimizado para largas sesiones de estudio nocturno.

## 📱 ¡Llévala en tu bolsillo! (PWA)

Esta aplicación es una **Progressive Web App (PWA)**. Esto significa que puedes instalarla en tu teléfono (Android o iOS) como si fuera una aplicación nativa:

1. Abre el enlace en el navegador de tu celular.
2. Busca la opción **"Instalar aplicación"** o **"Agregar a la pantalla de inicio"** en el menú del navegador.
3. ¡Listo! Tendrás un acceso directo con el logo de la UNEFA sin ocupar espacio excesivo.
